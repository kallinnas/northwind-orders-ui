## Project Setup Instructions

### Frontend (Angular)

1. **Clone the repository**:
    ```bash
    git clone https://github.com/kallinnas/northwind-orders-ui.git
    ```
2. **Navigate to the project directory**:
    ```bash
    cd northwind-orders-ui
    ```
3. **Install dependencies**:
    ```bash
    npm install
    ```
4. **Run the project**:
    ```bash
    ng serve
    ```
